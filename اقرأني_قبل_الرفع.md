# تحديث جديد على تطبيق حضور الطلاب

## ما الجديد في هذا التحديث؟

1. **إصلاح تصدير PDF على الموبايل**: تم استبدال طريقة "نافذة الطباعة" (التي لا تعمل جيداً على أغلب هواتف الأندرويد وآيفون خصوصاً عند تثبيت التطبيق) بتوليد ملف PDF حقيقي يُنزَّل مباشرة لجهازك — يعمل بنفس الطريقة على الكمبيوتر والموبايل.

2. **تصدير جديد: جدول التوقيتات**: زر "📋 تصدير جدول التوقيت للطلاب" في كل مجموعة — يصدّر PDF بسيط فيه فقط: اسم المجموعة، وقت البدء، وأسماء الطلاب وأقسامهم. مثالي لإرساله للطلاب عبر واتساب حتى يعرفوا موعد حضورهم.

## طريقة الرفع (تحديث الملفات الموجودة على GitHub)

بما أن عندك مستودع GitHub جاهز مسبقاً (`attendance-app`)، فقط:

1. افتح المستودع على GitHub
2. اضغط على ملف `index.html` الموجود
3. اضغط على أيقونة القلم (Edit) بأعلى يمين الصفحة
4. احذف كل المحتوى القديم، والصق محتوى ملف `index.html` الجديد المرفق هنا بالكامل
5. انزل للأسفل واضغط **Commit changes**
6. كرر نفس الخطوات لملف `service-worker.js` (تم تحديثه أيضاً لتخزين مكتبات PDF للعمل بدون إنترنت)

**ملاحظة:** تصدير PDF يحتاج إنترنت في أول استخدام فقط (لتحميل مكتبة توليد PDF)، وبعدها يعمل بدون إنترنت مثل بقية التطبيق.

بعد الرفع، أعد فتح رابط الموقع من الهاتف (قد تحتاج إغلاق التطبيق المثبّت وإعادة فتحه، أو حتى حذفه وإعادة تثبيته من الرابط ليتحدث بالكامل).
